# `ds-radio`

The `ds-radio` component is an interactive, presentational **controlled** Atom designed to render minimalist circular vector selection choices. It incorporates spring-physics hover/active transitions, full keyboard trigger listeners, validation fault styling, and root-level accessibility syncing.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host shell binds attributes (`selected`, `disabled`, `invalid`, `aria-label`) to toggle selection marks, suppress focus paths, or apply error indicators.
* **Events Out:** Captures user clicks or standard form interaction keys (`Space`, `Enter`) to internally toggle selection attributes and delegates native events upstream.

## Accessibility & Host ARIA Delegation

* **Aria Role Contract:** Assigns `role="radio"` and sets `tabindex="0"` on the custom element wrapper to comply with form control navigation specs.
* **Keyboard Navigation:** Full support for `Space` and `Enter` key presses.
* **Global Access Propagation:** Monitors the document root utilizing a `MutationObserver` to reflect accessibility class settings onto host attributes:
  * `.a11y-dark-mode`: Calibrates core colors for appropriate dark contrast.
  * `.a11y-high-contrast`: Enforces strict binary styling boundaries.
  * `.a11y-reduce-motion`: Disables centering scale animations.
  * `.a11y-forced-colors`: Employs Windows active system colors (`Highlight` for selected marks; `GrayText` for disabled elements).

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-radio';

const radio = document.createElement('ds-radio');
radio.setAttribute('selected', '');
radio.setAttribute('aria-label', 'Select primary theme layout');

// Clean sub-atomic overrides
radio.style.setProperty('--ds-radio-size', '20px');
radio.style.setProperty('--ds-radio-color', 'var(--color-success)');

radio.addEventListener('click', () => {
  console.log('Radio clicked. Selected status:', radio.hasAttribute('selected'));
});

document.body.appendChild(radio);
```
