## [1.0.0] - 2026-07-20

### Initial Release

The stable debut of the presentational vector radio selection badge (`ds-radio`) featuring spring-physics animations, full keyboard tab focusing, invalid state flags, and root observer accessibility syncing.

### Added
- **Interactive Radio Atom:** Created `<ds-radio>` acting as a form choice indicator with spring-physics transitions.
- **Form Integration States:** Equipped with observed `selected`, `disabled`, and `invalid` attributes that sync with native `aria-*` tags in real-time.
- **Root Class Syncing:** MutationObserver on HTML root elements reflects accessibility states as host boolean flags for styling.
- **Thematic Overrides:** Bound bounding container size and stroke/fill colors to `--ds-radio-size` and `--ds-radio-color` hooks.
- **Storybook Workspace:** Authored interactive controls and CSF3 story parameters in `Radio.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Modified Storybook custom style pipelines to write precise `--ds-radio-size` and `--ds-radio-color` tokens rather than generic variables.
