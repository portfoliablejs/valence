### Added
- **Initial Presentational Component:** Introduced `<ds-kbd>` built specifically as a stateless, controlled presentational atom.
- **Shortcut Combinations:** Integrated `show-plus` and `key` attributes to conditionally append combination syntax with the `+` character directly within shadow boundaries.
- **Root Accessibility Syncing:** Built-in `MutationObserver` on `document.documentElement` to reflect global visual states onto host attributes (`a11y-dark-mode`, `a11y-high-contrast`, `a11y-dyslexia`, `a11y-reduce-motion`, `a11y-forced-colors`).
- **Thematic Override Hooks:** Standardized variables for backdrop surfaces, radii, and border strokes using the `--ds-kbd-*` naming layout.
- **Storybook Integration:** Created interactive controls and CSF3 story parameters inside `Kbd.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Resolved broken style mappings in the Storybook render pipeline by binding props to the exact `--ds-kbd-radius`, `--ds-kbd-bg`, `--ds-kbd-border-width`, and `--ds-kbd-border-color` custom hooks.
