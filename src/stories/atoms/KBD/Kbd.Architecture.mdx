# `ds-kbd`

The `ds-kbd` component is a **presentational** Atom designed to render keyboard shortcuts, key combinations, and modifier cues within tooltips, command palettes, and inline text documentation.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host application binds modifier key labels through slot boundaries, and secondary key values through attributes (`variant`, `key`, `show-plus`) to configure combination display behavior.
* **Events Out:** As a purely passive presentational component, `ds-kbd` does not register external input bindings, trigger action-oriented events, or capture focus states.

## Slot Architecture

The `ds-kbd` component accepts standard text-based modifier labels or graphic glyphs in its default slot. When configured as a multi-key shortcut via the `show-plus` attribute, the internal layout engine appends a non-translatable `+` operator and an alphanumeric trailing key structure directly into the Shadow DOM container.

## Accessibility & Host ARIA Delegation

* **ARIA Semantics:** Uses passive host wrapper tags containing semantic `<kbd>` structures inside the Shadow DOM boundary for proper screen reader translation.
* **Global Accessibility Propagation:** Listens to class changes on `document.documentElement` using a `MutationObserver` to sync accessibility modes across the shadow boundary:
  * `.a11y-dark-mode`: Adjusts backdrop contrast scales.
  * `.a11y-high-contrast`: Installs binary `#000000` / `#FFFFFF` WCAG AAA boundaries.
  * `.a11y-dyslexia`: Enforces dyslexia-friendly typeface mappings and spacing.
  * `.a11y-forced-colors`: Applies system active `Canvas` and `CanvasText` palettes for Windows High Contrast Mode.
  * `.a11y-reduce-motion`: Forces standard zero-duration animation transitions.

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-kbd';

const kbd = document.createElement('ds-kbd');
kbd.textContent = '⌥ Option';
kbd.setAttribute('show-plus', '');
kbd.setAttribute('key', 'P');

// Overriding default style tokens cleanly
kbd.style.setProperty('--ds-kbd-radius', '4px');
kbd.style.setProperty('--ds-kbd-bg', 'var(--color-gray-xlight)');

document.body.appendChild(kbd);
```
