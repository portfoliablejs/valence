# `ds-subtitle`

The `ds-subtitle` component is an interactive, presentational **controlled** Atom designed for overlay media captions and real-time screen reader announcements. It supports free pointer-dragging repositioning, incorporates CSS-driven fade animations, and implements polite ARIA announcements.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host shell binds properties (`text`, `visible`, `aria-label`) to render caption strings and trigger visibility transitions.
* **Events Out:** Captures mouse/pointer events inside the custom element to support free drag coordinate mapping, dispatching tracking events (`ds-subtitle-dragstart`, `ds-subtitle-dragend`) upstream.

## Accessibility & Host ARIA Delegation

* **Aria Role Contract:** Incorporates `aria-live="polite"` directly on internal subtitle wrappers, prompting screen readers to immediately capture VTT cue updates without breaking page context.
* **ARIA Host Scrubbing:** Automatically scrubs the Light DOM host attribute `aria-label` and delegates it down to the internal text container.
* **Global Accessibility Propagation:** Integrates `_observeRootAccessibility` to sync viewport visual preferences:
  * `.a11y-dark-mode`: Shifts background colors to dark media player HUD tokens.
  * `.a11y-high-contrast`: Enforces strict AAA binary contrast ratios (thick black borders with solid white backgrounds).
  * `.a11y-dyslexia`: Forces Comic Sans MS fonts for dyslexia accommodations.
  * `.a11y-reduce-motion`: Forces instant 0ms visibility transitions.
  * `.a11y-forced-colors`: Employs Windows active system colors (`Canvas` and `CanvasText`).

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-subtitle';

const subtitle = document.createElement('ds-subtitle');
subtitle.setAttribute('text', 'Hello, and welcome to this case study!');
subtitle.setAttribute('visible', 'true');
subtitle.setAttribute('aria-label', 'English captions');

// Clean sub-atomic overrides
subtitle.style.setProperty('--ds-subtitle-bg', 'rgba(0, 0, 0, 0.85)');
subtitle.style.setProperty('--ds-subtitle-color', 'yellow');

document.body.appendChild(subtitle);
```
