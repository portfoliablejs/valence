# Changelog

<br />

## [1.0.0] - 2026-07-20

<br />

### Initial Release

The stable debut of the presentational caption overlay component (`ds-subtitle`) featuring interactive pointer-based viewport dragging, polite screen reader captions, and responsive CSS scale modifiers.

### Added
- **Caption Overlay Atom:** Created `<ds-subtitle>` acting as a media caption HUD.
- **Pointer Dragging repositioning:** Built-in pointerdown/pointermove/pointerup handlers supporting custom drag triggers.
- **Thematic Overrides:** Standardized backdrop colors, font stacks, container padding, corner radii, and outline widths under `--ds-subtitle-*` layouts.
- **Storybook Workspace:** Configured comprehensive CSF3 stories and automated interactions inside `Subtitle.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Synced Storybook template controls to pass accurate `--ds-subtitle-*` style overrides.
