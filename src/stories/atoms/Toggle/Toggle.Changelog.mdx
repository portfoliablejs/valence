## [1.0.0] - 2026-07-20

<br />

### Initial Release

The stable debut of the presentational binary and text toggle switch (`ds-toggle`) featuring hardware-accelerated pointer drag tracking, dynamic text label width adjustments, and root class accessibility observing.

### Added
- **Pill Switch Atom:** Created `<ds-toggle>` serving as a binary toggle button.
- **Hardware-Accelerated Drag Tracking:** Interactive pointerdown/pointermove/pointerup event listeners enabling smooth sliding knob translations.
- **Root Class Syncing:** MutationObserver on document roots reflects global setting cascades downstream as host element boolean flags.
- **Thematic Overrides:** Bound track size, knob size, border radius, background, checked background, border-width, border-color, and knob-bg variables to `--ds-toggle-*` custom properties.
- **Storybook Workspace:** Configured complete interactive Controls and detailed CSF3 stories inside `Toggle.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Synchronized Storybook override controllers to output precise `--ds-toggle-checked-bg` custom styles.
