## [1.0.0] - 2026-07-16

<br />

### Initial Release

The stable debut of the presentational vector iconography component (`ds-icon`) featuring comprehensive media controls, outline/fill vector variants, automatic RTL directionality flipping, and forced color overrides.

### Added
- **Unified Icon Registry:** Introduced `<ds-icon>` built specifically with outline/fill variants, and a comprehensive central `ICON_REGISTRY` covering media playback, volume, and navigation keys.
- **Directional Mirroring:** Built-in X-axis scale flipping `transform: scaleX(-1)` automatically triggered when `dir="rtl"` is mutated on the HTML root element.
- **Root Context Syncing:** MutationObserver on document roots reflects global setting cascades downstream as host element boolean flags.
- **Thematic Overrides:** Bound bounding box dimensions, stroke weight, fill color, and outline colors to standard `--ds-icon-*` variables.
- **Storybook Workspace:** Configured complete interactive Controls and detailed CSF3 stories inside `Icon.stories.js`.

### Fixed
- **Screen Reader Announce Prevention:** Enforces `aria-hidden="true"` on inner SVG elements to guarantee decorative presentational isolation.
- **Container Text Color Inheritance:** Sets host `color: inherit` ensuring icons cleanly inherit typography colors from parent wrappers.
