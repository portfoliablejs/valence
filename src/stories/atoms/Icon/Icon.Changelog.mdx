# Changelog
<br />
## [1.0.0] - 2026-07-16
<br />
### Initial Release

### Added
- **Initial Component Implementation:** First stable release of `<ds-icon>` featuring:
  - **System Icon Registry:** Unified `ICON_REGISTRY` offering comprehensive media controls, volume gauges, navigation chevrons, user actions, quality badges, and system utility icons.
  - **Variant Controls:** Dual-mode vector path support for `outline` and `fill` variants.
  - **Directional RTL Mirroring:** Automatic horizontal flipping (`transform: scaleX(-1)`) for navigation and directional icons when `dir="rtl"` is detected on the root document.
  - **Sub-Atomic Token Hooks:** Component-specific CSS custom property overrides for sizing, stroke weight, fill, and stroke color (`--ds-icon-size`, `--ds-icon-stroke-width`, `--ds-icon-fill`, `--ds-icon-stroke`).
  - **Root Context Observer (`_observeRootContext`):** Integrated `MutationObserver` listening to root document attribute and class changes to synchronize RTL directionality, high contrast, and forced colors.
  - **WCAG AAA & Forced Colors Support:** Complete Windows Forced Colors Mode compatibility utilizing system `CanvasText` and `currentColor` mappings.
- **Storybook CSF3 Integration:** Story controls in `Icon.stories.js` featuring range controls for stroke weight and size, variant selectors, and container color inheritance validation.

### Fixed
- **Screen Reader Announce Prevention:** Enforces `aria-hidden="true"` on inner SVG elements to guarantee decorative presentational isolation.
- **Container Text Color Inheritance:** Sets host `color: inherit` ensuring icons cleanly inherit typography colors from parent wrappers.
