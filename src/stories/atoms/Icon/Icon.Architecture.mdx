# `ds-icon`

The `ds-icon` component is a **stateless / presentational** Atom designed to render vector graphics from a standardized system iconography registry, supporting outlined and filled variants, automatic RTL flipping, and high-contrast system overrides.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host application binds primitive attributes (`name`, `variant`, `size`, `color`) or sub-atomic custom CSS property overrides onto the host element.
* **Registry Resolution:** `ds-icon` queries `ICON_REGISTRY` using the `name` attribute and renders the matching outline or fill SVG path markup inside the Shadow DOM.
* **Encapsulated Presentation:** SVG viewbox scaling, stroke geometry, and theme mode overrides are encapsulated inside the Shadow DOM.

## Accessibility & Host ARIA Delegation

* **Decorative SVGs:** The internal `<svg>` primitive always renders with `aria-hidden="true"` to prevent screen readers from announcing decorative graphics.
* **Directional RTL Mirroring:** Listens to `dir="rtl"` on `document.documentElement` via `MutationObserver` and applies `[dir-rtl]` to mirror directional icons (e.g., chevrons, arrows, volume, and playback controls) along the X-axis using `transform: scaleX(-1)`.
* **Global Accessibility Propagation:** Observes `document.documentElement` class changes to handle high contrast and forced color modes:
  * `.a11y-high-contrast`: Forces vector stroke to inherit `currentColor` and strips background fills.
  * `.a11y-forced-colors`: Complies with OS-level Forced Colors Mode by mapping strokes to `currentColor` and filled variants to `CanvasText`.

## Examples

### Basic Integration Example

```javascript
import 'valence/Iconography.js';

// Search Icon (Outline)
const searchIcon = document.createElement('ds-icon');
searchIcon.setAttribute('name', 'search');
searchIcon.setAttribute('size', '24');

// Bookmark Icon (Filled Variant with Sub-Atomic Overrides)
const bookmarkIcon = document.createElement('ds-icon');
bookmarkIcon.setAttribute('name', 'bookmark');
bookmarkIcon.setAttribute('variant', 'fill');
bookmarkIcon.style.setProperty('--ds-icon-size', '32px');
bookmarkIcon.style.setProperty('--ds-icon-stroke-width', '1.5');

document.body.appendChild(searchIcon);
document.body.appendChild(bookmarkIcon);
```
