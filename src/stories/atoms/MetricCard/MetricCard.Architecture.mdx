# `ds-metric-card`

The `ds-metric-card` component is a **stateless (presentational)** Atom designed for displaying metric figures, key performance indicators (KPIs), directional trends, and summary statistics across portfolio case studies and dashboard cards.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **Unidirectional Data Flow** pattern:
* **State In (`value`, `label`, `prefix`, `suffix`, `variant`, `treatment`, `size`, `trend`, `icon`):** The host application binds primitive attributes downstream onto the host element tag.
* **Passive Presentation:** The component contains zero business logic, performs no internal calculations, and renders strictly what is passed downstream.

## Accessibility & Host ARIA Delegation

* **ARIA Scrubbing:** Reads explicit `aria-label` or auto-generates composite accessible reading labels (`${prefix}${value}${suffix} ${label}`) and delegates them to the inner Shadow DOM container.
* **Global Accessibility Propagation:** Listens to class changes on `document.documentElement` via `MutationObserver` to sync accessibility modes:
  * `.a11y-dark-mode`: Adjusts border, background, and text contrast levels.
  * `.a11y-high-contrast`: Enforces thick 3px white border indicators.
  * `.a11y-large-text`: Scales metric value and label font sizes.
  * `.a11y-dyslexia`: Applies high-legibility letter spacing and font stacks.
