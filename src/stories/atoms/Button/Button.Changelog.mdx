# Changelog

<br />

## [1.0.0] - 2026-07-16

<br />

### Initial Release

The stable debut of the presentational button component (`ds-button`) featuring support for nesting icons, custom tooltips, multiple stylistic form factor variants, and active root accessibility syncs.

### Added
- **Core Variants:** Introduced `<ds-button>` built specifically with five functional variant structures (`primary`, `secondary`, `text`, `icon`, `floating`).
- **Iconography Synergy:** Auto-calculates icon dimensions based on active variant templates and reverses directions when `icon-position="right"` is active.
- **Root Class Syncing:** Built-in `MutationObserver` on the HTML root reflects global settings onto host element attributes.
- **Thematic Overrides:** Bound border-radius, background, hover backdrop, active backdrop, borders, and viewport scales to standard `--ds-button-*` properties.
- **Storybook Workspace:** Authored interactive Controls and automated CSF3 story play-tests inside `Button.stories.js`.

### Fixed
- **Host ARIA Delegation:** Clears light DOM `aria-label` custom attributes on host custom elements after delegating them to focusable internal buttons.
- **Disabled Host Cursor:** Enforces `cursor: not-allowed` on the host wrapper element when disabled is mapped.
