### Added
- **Selection Indicator Atom:** Created `<ds-check>` serving as an animated vector completion checkmark.
- **Surface Containers:** Smooth scale-opacity check transitions with optional background framing when `has-background` is active.
- **Root Class Syncing:** MutationObserver on document roots reflects global setting cascades downstream as host element boolean flags.
- **Thematic Overrides:** Bound outer dimensions, stroke vector color, background container fill, and border ring colors to the standard `--ds-check-*` naming layout.
- **Storybook Workspace:** Configured complete interactive Controls and detailed CSF3 stories inside `Check.stories.js`.

### Fixed
- **Shadow DOM Layout Nodes:** Compressed inner Shadow DOM HTML templates on single lines to resolve ghost whitespace layout nodes.
- **Keyboard & Disabled Focus Handling:** Suppressed keyboard spacebar/enter triggers and automatically assigned `tabindex="-1"` when the `disabled` attribute is present.
