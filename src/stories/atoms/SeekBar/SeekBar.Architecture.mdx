# `ds-seek-bar`

The `ds-seek-bar` component is an interactive, highly accessible **controlled** Atom slider designed for media timelines, volume controllers, and custom playback mixers. It is backed by a mouse-tracking `ds-tooltip` molecule, incorporates spring-physics animations, and supports right-to-left layout orientations.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host binds properties (`percent`, `variant`, `disabled`, `show-tooltip`, `tooltip-text`, `show-kbd`, `kbd-label`, `aria-label`) downstream to configure timeline positions, track variants, or customize tooltip indicators.
* **Events Out (`ds-seek`):** Dispatch custom events containing the current percent calculation when users drag, touch, or click along the active container bounds.

## Slot Architecture

The `ds-seek-bar` component is structurally sealed, wrapping its tracks, fills, handle knobs, and floating tooltip elements directly inside its Shadow DOM boundaries. No public slot targets are exposed to ensure robust click-target calculations.

## Accessibility & Host ARIA Delegation

* **Aria Role Contract:** Delegates `role="slider"`, `aria-valuemin="0"`, and `aria-valuemax="100"` to the internal sliding container.
* **Keyboard Navigation:** Fully supports `ArrowRight`, `ArrowUp` (increment), `ArrowLeft`, `ArrowDown` (decrement), `PageUp` (+10%), `PageDown` (-10%), `Home` (0%), and `End` (100%) interactions.
* **Bi-directional Support:** Automatically tracks root-level `dir="rtl"` mutations on `document.documentElement` to invert dragging orientations and key translations.
* **Global Access Syncing:** Employs a `MutationObserver` to toggle accessibility classes as boolean host attributes:
  * `.a11y-dark-mode`: Adjusts default track contrast boundaries.
  * `.a11y-high-contrast`: Enforces strict binary tracks and bordered knobs.
  * `.a11y-reduce-motion`: Disables track expansion transitions.
  * `.a11y-forced-colors`: Employs Windows active system colors (`Highlight` for fills; `HighlightText` for knobs; `CanvasText` for tracks).

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-seek-bar';

const seek = document.createElement('ds-seek-bar');
seek.setAttribute('percent', '35');
seek.setAttribute('aria-label', 'Timeline seeker');
seek.setAttribute('show-kbd', '');
seek.setAttribute('kbd-label', 'Space');

// Custom design token overrides
seek.style.setProperty('--ds-seek-bar-track-bg', 'rgba(255, 255, 255, 0.1)');
seek.style.setProperty('--ds-seek-bar-fill-bg', 'var(--color-success)');

seek.addEventListener('ds-seek', (e) => {
  console.log('User seeking to:', e.detail.percent);
});

document.body.appendChild(seek);
```
