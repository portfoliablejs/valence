## [1.0.0] - 2026-07-20

### Initial Release

The stable debut of the presentational media timeline seek-bar slider (`ds-seek-bar`) featuring custom tooltips, RTL support, keyboard navigation keys, and accessibility syncing.

### Added
- **Timeline Slider Atom:** Created `<ds-seek-bar>` designed for media players and volume mixers.
- **Bi-directional Support:** Dynamically tracks root-level `dir` attribute and adjusts dragging calculations and key navigation offsets.
- **Root Preference Observing:** Built-in `MutationObserver` on the HTML document root reflects global settings as host element boolean flags.
- **Thematic Override Hooks:** Bound track background, fill progress, and handle knob colors to `--ds-seek-bar-track-bg`, `--ds-seek-bar-fill-bg`, and `--ds-seek-bar-knob-bg`.
- **Storybook Workspace:** Built comprehensive interactive Controls and detailed CSF3 stories inside `SeekBar.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Synchronized Storybook override controllers to output precise `--ds-seek-bar-*` custom styles.
