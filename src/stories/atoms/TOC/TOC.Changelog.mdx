# Changelog

<br />

## [1.1.0] - 2026-07-22

<br />

### WebKit Layer Isolation & CONTENTS Header Standard

Synchronized `ds-toc` with the `ds-contextual-menu` v1.1.0 scrollbar refactor. Updated default popover title headers to `'CONTENTS'`, linked sub-atomic height tokens to inner scroll viewports, and resolved WebKit layer freeze bugs when opening the minimap overlay.

<br />

### Refactored
- **Default Header Title:** Configured `updateAttributes()` to default `title-text` to `'CONTENTS'`, ensuring popover menus render clear header titles unless explicitly overridden.
- **Height Token Mapping:** Linked `--ds-toc-max-height` to both `--ds-contextual-menu-max-height` and `--ds-contextual-menu-scroll-max-height` inside `toc.css`, ensuring long lists clip properly at custom viewport boundaries.

### Fixed
- **Safari Layer Freeze on Reveal:** Added `will-change: opacity, transform` to `.toc-menu` and attached a micro-layout flush (`void offsetHeight`) inside `attributeChangedCallback` when `opened` switches to `true`, forcing WebKit to re-bind nested scrollbar composite layers cleanly.

### Added
- **13-Item Overflow Testing:** Updated standard mock heading pools to 13 default items and introduced the `ThirteenItemsOverflow` story with automated play interaction tests.

## [1.0.0] - 2026-07-20

<br />

### Initial Release

The stable debut of the Table of Contents Atom (`ds-toc`) featuring an Apple-like minimap strip, automatic DOM heading scanning (`<h1>`–`<h4>`), `IntersectionObserver` section tracking, and contextual popover menu integration.
