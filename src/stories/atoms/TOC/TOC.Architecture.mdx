# `ds-toc`

The `ds-toc` component is a **controlled** Atom designed to render an Apple-like document minimap strip that expands into a floating navigation overlay (`ds-contextual-menu`). It automatically scans document heading elements (`<h1>`–`4>`), monitors scrolling via `IntersectionObserver`, highlights active content sections, and provides smooth programmatic scroll navigation.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host binds `opened`, `title-text`, `target-selector`, `max-height`, `aria-label`, or programmatically injects an array of `items` heading objects downstream.
* **Events Out:** Selecting a section heading dispatches a composed `ds-toc-select` custom event with heading metadata. Selecting the "Scroll to top" option dispatches `ds-toc-scroll-top`.

## Slot Architecture

The `ds-toc` component completely encapsulates its internal DOM structure. It projects no public `<slot>` elements to ensure the Apple-style minimap strip and popover menu maintain strict positioning alignment and padding rules.
