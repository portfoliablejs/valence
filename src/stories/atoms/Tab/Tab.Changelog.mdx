# Changelog

## [1.0.0] - 2026-07-20

<br />

### Initial Release

The stable debut of the presentational navigation control trigger (`ds-tab`) featuring native button integration, high-contrast states, custom click events, and root class accessibility syncing.

### Added
- **Interactive Tab Trigger Atom:** Created `<ds-tab>` as a stateless, controlled pagination button.
- **Robust Accessibility:** Features keyboard tab paths, aria-selected and tabindex reflections, and proper light DOM aria-label scrubbing.
- **Root Preference Synchronization:** MutationObserver on document roots reflects global setting cascades downstream as boolean host flags.
- **Thematic Overrides:** Bound diameter dimensions, border-radius, background fill, active background, and hover background color tokens to the standardized `--ds-tab-*` naming layout.
- **Storybook Workspace:** Authored interactive controls and CSF3 story parameters in `Tab.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Synchronized Storybook override controllers to output precise `--ds-tab-size`, `--ds-tab-bg`, and `--ds-tab-active-bg` custom styles.
