# `ds-tab`

The `ds-tab` component is an interactive, presentational **controlled** Atom designed to render tab triggers inside tablist widgets. It includes native key event hooks, supports dual slot/attribute label configuration, handles disabled states, and implements root class accessibility observer syncs.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host binds attributes (`label`, `active`, `disabled`, `aria-label`) to render label contents, toggle selection highlights, or block interaction events.
* **Events Out (`ds-tab-click`):** Captures mouse clicks to dispatch unified cross-boundary events indicating which tab index has been focused and triggered.

## Slot Architecture

The `ds-tab` component exposes its default slot to receive raw strings or HTML tags for visual layout labels. When a direct `label` attribute is defined on the host custom element, it completely overrides the projected slot contents.

## Accessibility & Host ARIA Delegation

* **Semantics & Focusing:** Employs an internal `<button>` with `role="tab"` to handle focus paths correctly. Tab index is set to `0` (active) and `-1` (inactive) for standard keyboard focus flows.
* **ARIA Host Scrubbing:** Automatically purges Light DOM `aria-label` tags and projects them cleanly onto the focusable button.
* **Global Access Syncing:** Uses a `MutationObserver` on the HTML root to synchronize visual settings:
  * `.a11y-dark-mode`: Swaps inactive font palettes and maps active containers to pure white.
  * `.a11y-high-contrast`: Enforces clear AAA solid fills with inverted text.
  * `.a11y-dyslexia`: Forces Comic Sans MS font overrides.
  * `.a11y-reduce-motion`: Disables transitions.
  * `.a11y-forced-colors`: Overrides backgrounds via system `Canvas` and selected fills via `Highlight`.

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-tab';

const tab = document.createElement('ds-tab');
tab.setAttribute('label', 'Analytics HUD');
tab.setAttribute('active', 'true');
tab.setAttribute('aria-label', 'View system statistics');

// Clean sub-atomic overrides
tab.style.setProperty('--ds-tab-radius', '4px');
tab.style.setProperty('--ds-tab-active-bg', 'var(--color-success)');

tab.addEventListener('ds-tab-click', (e) => {
  console.log('Tab selected. Value:', e.detail.label);
});

document.body.appendChild(tab);
```
