# Changelog
<br />
## [1.0.0] - 2026-07-16
<br />
### Initial Release

### Added
- **Initial Component Implementation:** First stable release of `<ds-divider>` featuring:
  - **Structural Orientations:** Support for both `horizontal` and `vertical` layout axes with standard flexible container sizing.
  - **Sub-Atomic Token Hooks:** Component-specific CSS custom property overrides for stroke color, rule thickness, and spatial margins (`--ds-divider-color`, `--ds-divider-thickness`, `--ds-divider-margin`).
  - **Root Accessibility Observer (`_observeRootAccessibility`):** Integrated `MutationObserver` on `document.documentElement` to automatically sync global accessibility modes onto host custom element attributes.
  - **WCAG AAA & Forced Colors Support:** High-contrast overrides, reduced motion suppression, dark mode contrast mapping, and Windows Forced Colors Mode system palette mapping (`CanvasText`).
- **Storybook CSF3 Integration:** Story controls in `Divider.stories.js` featuring sub-atomic knobs (`color`, `thickness`, `margin`) and automated interaction testing via `play` functions validating host attribute scrubbing and internal ARIA propagation.

### Fixed
- **Host ARIA Delegation & Infinite Reflection Guard:** Automatically scrubs `aria-label` from the host custom element tag after delegating it to the inner `.divider-root` primitive, with a guard in `attributeChangedCallback` preventing infinite loop reflections when `aria-label` is removed.
- **Shadow DOM Layout Nodes:** Compressed inner HTML template string into a single line to eliminate anonymous whitespace text nodes in Shadow DOM layouts.

*This changelog is specific to the `<ds-divider>` component. For design system global releases, refer to the main [Getting Started/Changelog](?path=/docs/getting-started-changelog--docs).*
