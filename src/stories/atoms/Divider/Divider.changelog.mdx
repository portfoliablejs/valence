# Changelog

## [1.0.0] - 2026-07-16

<br />

### Initial Release

The stable debut of the presentational content divider component (`ds-divider`) featuring support for horizontal and vertical layouts, customizable thicknesses and colors, and active root accessibility syncs.

### Added
- **Structural Divider Atom:** Created `<ds-divider>` designed for separating layout blocks.
- **Bi-directional Support:** Dynamically delegates `aria-orientation` and `aria-label` downstream to role="separator" nodes.
- **Root preference Syncing:** MutationObserver on document roots reflects global setting cascades downstream as host element boolean flags.
- **Thematic Overrides:** Bound separator line color, thickness, and spatial margins to `--ds-divider-color`, `--ds-divider-thickness`, and `--ds-divider-margin`.
- **Storybook Workspace:** Configured complete interactive Controls and detailed CSF3 stories inside `Divider.stories.js`.

### Fixed
- **Host ARIA Delegation & Infinite Guard:** Installs safe `attributeChangedCallback` blocks preventing infinite recursion when purging Light DOM host attributes.
- **Shadow DOM Layout Nodes:** Compressed inner Shadow DOM HTML templates on single lines to resolve ghost whitespace layout nodes.
