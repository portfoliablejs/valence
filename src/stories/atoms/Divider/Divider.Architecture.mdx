# `ds-divider`

The `ds-divider` component is a **stateless / presentational** Atom designed to separate content blocks horizontally or vertically while handling semantic accessibility delegation and sub-atomic custom token overrides.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host application binds layout attributes (`orientation`, `aria-label`) or custom CSS property overrides onto the host element.
* **Internal Delegation:** `ds-divider` delegates `aria-label` and `aria-orientation` attributes down to its internal Shadow DOM primitive (`role="separator"`) and scrubs `aria-label` from the host element to eliminate redundant ARIA warnings.
* **Encapsulated Presentation:** Layout geometry, margin boundaries, and color transitions are encapsulated inside the Shadow DOM.

## Accessibility & Host ARIA Delegation

* **Role & ARIA Scrubbing:** The internal primitive carries `role="separator"`. Passing an `aria-label` to `<ds-divider>` delegates `aria-label` and `aria-orientation` directly to the internal `.divider-root` element and automatically purges `aria-label` from the host custom element tag.
* **Global Accessibility Propagation:** Listens to class changes on `document.documentElement` via `MutationObserver` to sync accessibility modes onto host custom element attributes:
  * `.a11y-dark-mode`: Adjusts rule line stroke contrast against dark surfaces.
  * `.a11y-high-contrast`: Enforces strict binary black stroke bounds.
  * `.a11y-forced-colors`: Maps system active `CanvasText` palette in Windows High Contrast Mode.
  * `.a11y-reduce-motion`: Suppresses background color transition durations.

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-divider';

// Horizontal Section Separator
const horizontalDivider = document.createElement('ds-divider');
horizontalDivider.setAttribute('orientation', 'horizontal');
horizontalDivider.setAttribute('aria-label', 'Main section break');

// Vertical Content Separator with Sub-Atomic Overrides
const verticalDivider = document.createElement('ds-divider');
verticalDivider.setAttribute('orientation', 'vertical');
verticalDivider.style.setProperty('--ds-divider-color', 'var(--color-accent)');
verticalDivider.style.setProperty('--ds-divider-thickness', '2px');

document.body.appendChild(horizontalDivider);
document.body.appendChild(verticalDivider);
```
