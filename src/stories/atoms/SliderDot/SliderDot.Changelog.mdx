### Initial Release

The stable debut of the presentational pagination indicator (`ds-slider-dot`) featuring native button integration, high-contrast layouts, custom interaction click events, and root class accessibility syncing.

### Added
- **Interactive Pagination Atom:** Created `<ds-slider-dot>` as a stateless, controlled pagination button.
- **Robust Accessibility:** Features keyboard tab paths, spacebar trigger listeners, aria-current reflections, and proper light DOM aria-label scrubbing.
- **Root Preference Synchronization:** MutationObserver on document roots reflects global setting cascades downstream as boolean host flags.
- **Thematic Overrides:** Bound diameter dimensions, border-radius, background fill, active background, and hover background color tokens to the standardized `--ds-slider-dot-*` naming layout.
- **Storybook Workspace:** Authored interactive controls and CSF3 story parameters in `SliderDot.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Synchronized Storybook override controllers to output precise `--ds-slider-dot-size`, `--ds-slider-dot-bg`, and `--ds-slider-dot-active-bg` custom styles.
