# Changelog

<br />

## [1.0.0] - 2026-07-23

<br />

### Minimal Motion Engine Prop

This release makes the gallery motion profile configurable through an `engine` prop while preserving the current minimal tuning as the default. The gallery keeps its restrained, minimalistic interaction model, but the motion contract is now explicit and reusable.

### Changed
- **Motion Contract:** The gallery now derives drag behavior from a named motion engine instead of fixed inline tuning values.
- **Default Preset:** The `minimal` engine remains the default so the gallery keeps its restrained feel out of the box.
- **Storybook Surface:** Story controls now include the motion engine prop alongside item count and gap spacing.

### Added
- **Architecture Documentation:** Added a component architecture document covering state flow, API contract, accessibility, and host integration.
- **Changelog Documentation:** Added release notes for the gallery organism so future motion updates can be tracked in one place.

### Fixed
- **Reduced Motion Compliance:** Added a no-motion path that disables the gallery's shrink animation and animated edge-return behavior when `.a11y-reduce-motion` is active.
