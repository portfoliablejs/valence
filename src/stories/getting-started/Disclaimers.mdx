## 4. Guidelines for Portfolio Builders & End Users

When using this component inside an open-source portfolio builder, Web application, or exported static site:

1. **User Content Responsibility:** End users publishing portfolios are solely responsible for ensuring that the content, images, and artwork displayed inside the `screen-image` layer do not infringe upon any third-party copyrights or trademarks.
2. **Commercial Distribution Notice:** If you choose to bundle or sell exported portfolio themes containing device mockup assets, verify that your redistribution complies with local trademark laws regarding commercial hardware depictions.
3. **Opt-Out / Custom-Only Usage:** Users wishing to avoid displaying trademarked physical bezels entirely can enable the `custom-only` attribute on `<ds-thumbnail custom-only>` to hide all device overlays while preserving clean, unbranded container styling.
