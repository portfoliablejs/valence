# Portfoliable Application Integration Guide

The `portfoliable` application is an open-source, web-native portfolio builder designed as a single-page Web Component shell (`AppShell`). It dynamically consumes, orchestrates, and renders templates from the **Valence Design System** (`ds-*` Web Components).

This document serves as the primary integration guide explaining how the host application interacts with Valence components, passes data through shadow boundaries, manages global accessibility cascades, and handles runtime state transitions.

<br />

## 1. Application Architecture & Component Shell

The application entrypoint (`src/main.js`) mounts the root Web Component (`<app-shell>`) directly into the DOM viewport. `AppShell` encapsulated inside `src/App.js` maintains application state and renders Valence components into dedicated view containers (`#view-home`, `#view-case`, `#view-about`).

### B. Shadow DOM Slot Injection
For complex, formatted content (such as rich case study bodies, code snippets, or inline telemetry cards), `AppShell` injects HTML subtrees directly into component `<slot>` targets:

```javascript
// Example: Injecting rich localized HTML into <ds-case-slide> slot
const slide = document.createElement('ds-case-slide');
slide.dataset.caseId = caseData.id;
slide.setAttribute('title', this.getLang(caseData.title));

// Choose executive vs recruiter content based on app state
const descContent = this.state.isRecruiterMode ? caseData.descRecruiter : caseData.desc;

// Slot injection target
slide.innerHTML = `<div slot="description">${this.getLang(descContent)}</div>`;

```

## 3. Internationalization (`i18n`) Pipeline

Multilingual support is managed via a lightweight `i18n.js` module and localized data schemas in `data.js`.

### Multilingual Data Definition Schema

Case study records define string attributes using a two-locale dictionary helper:

```javascript
// src/data.js
const i18n = (en, pt) => ({ en, pt });

export const holofante = {
  id: "holofante",
  title: i18n("Agentic AI Design", "Design de IA Agêntica"),
  shortDesc: i18n("DesignOps automation cuts delivery latency by 70%.", "Automação de DesignOps reduz a latência de entrega em 70%."),
  readTime: i18n("3 min", "3 min"),
  // ...
};

```

### Runtime Translation Resolution

When rendering Valence components, `AppShell` invokes `this.getLang(property)` to evaluate the user's active locale (`window.currentLang`):

```javascript
// Language resolution helper inside AppShell
getLang(prop) {
  if (prop && prop[this.state.lang]) {
    return prop[this.state.lang];
  }
  return (prop && prop['en']) ? prop['en'] : prop;
}

```

## 4. Dual-Persona Content Injection (Executive vs. Recruiter)

To serve both executive hiring managers and technical recruiters, `portfoliable` dynamically mutates the content slotted into `<ds-case-slide>` based on `this.state.isRecruiterMode`:

```

│                             ds-header                                    │
│  [Executive Mode] <----------------------------------> [Recruiter Mode]  │
└────────────────────────────────────┬─────────────────────────────────────┘
                                     │
                             ds-mode-change
                                     │
                                     v
                       AppShell.setState({ isRecruiterMode })
                                     │
            ┌────────────────────────┴────────────────────────┐
            ▼                                                 ▼
   [Executive View: desc]                          [Recruiter View: descRecruiter]
   - Business Impact Metrics                       - System Architecture Flowcharts
   - User Experience Storytelling                  - Client Runtime Memory Models
```

When switching modes, `AppShell` clears and re-populates the `#case-slider-container` children, swapping the slotted HTML template passed into `<ds-case-slide>` without altering component wrapper attributes.

## 5. Global Accessibility (`a11y`) Propagation Engine

Accessibility settings configured inside `<ds-modal id="a11y-modal">` are stored in `localStorage` and synchronized across the entire DOM root.

### State Mutation & Class Cascade

When a user toggles an accessibility setting, `_handleA11yChange()` mutates `this.state.a11y` and calls `applyA11ySettings()`:

```javascript
// Reflects a11y preferences onto document.documentElement as root classes
applyA11ySettings() {
  const htmlEl = document.documentElement;
  const settings = this.state.a11y;
  
  Object.entries(settings).forEach(([key, value]) => {
    // Converts camelCase to kebab-case (e.g. darkMode -> a11y-dark-mode)
    const className = `a11y-${key.replace(/([A-Z])/g, '-$1').toLowerCase()}`;
    htmlEl.classList.toggle(className, value);
  });
}

```

### Valence Component Shadow DOM Observation

Every Valence Web Component executes `_observeRootAccessibility()` during `connectedCallback()`. A `MutationObserver` monitors `document.documentElement` for class changes and reflects boolean attributes onto the component host tag:

```javascript
// Inside every Valence component class (e.g., Button.js, Header.js)
_observeRootAccessibility() {
  const root = this.ownerDocument.documentElement;
  const sync = () => {
    this.toggleAttribute('a11y-dark-mode', root.classList.contains('a11y-dark-mode'));
    this.toggleAttribute('a11y-high-contrast', root.classList.contains('a11y-high-contrast'));
    this.toggleAttribute('a11y-large-text', root.classList.contains('a11y-large-text'));
    this.toggleAttribute('a11y-dyslexia', root.classList.contains('a11y-dyslexia'));
    this.toggleAttribute('a11y-reduce-motion', root.classList.contains('a11y-reduce-motion'));
    this.toggleAttribute('a11y-forced-colors', root.classList.contains('a11y-forced-colors'));
  };

  sync();
  this._themeObserver = new MutationObserver(sync);
  this._themeObserver.observe(root, { attributes: true, attributeFilter: ['class'] });
}

```

This decoupled observer pattern ensures that all active design system components update their visual styling instantly without requiring full-page re-renders or main-thread state cycles.
