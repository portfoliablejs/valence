# `ds-case-navigator`

The `ds-case-navigator` component is a **composite** Molecule designed for sequential case study navigation, inline autocomplete search filtering, and global keyboard shortcut orchestration.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host application binds current index position, total case count, search queries, and localized label maps.
* **Events Out (`ds-*`):** Navigation triggers dispatch custom DOM events (`ds-case-select`, `ds-case-prev`, `ds-case-next`, `ds-search-input`, `ds-search-select`) containing immutable detail payloads.
* **Encapsulated State:** The component maintains internal highlight index positioning for autocomplete filtering without mutating host router state directly.

## Accessibility & Host ARIA Delegation

* **ARIA Scrubbing & Valid Attribute Names:** Generic `aria-label` on host elements is scrubbed and delegated to internal controls. Custom accessibility label props use valid attribute names (`search-aria-label`, `prev-aria-label`, `next-aria-label`) to comply with strict W3C validation.
* **Dynamic Role Management:** The autocomplete container dynamically attaches `role="listbox"` and `aria-label="Search suggestions"` only when option items are present, preventing `aria-prohibited-attr` and `aria-required-children` failures when hidden.
* **Keyboard Navigation:** Full support for `Space` and `Enter` execution on buttons. Global shortcut listeners manage single-key (`S`, `X`, `←`, `→`) and multi-key combinations (`Cmd+F`, `Cmd+←`, `Cmd+→`).
* **Global Accessibility Propagation:** Listens to class changes on `document.documentElement` via `MutationObserver` to sync accessibility modes (`.a11y-dark-mode`, `.a11y-high-contrast`, `.a11y-large-text`, `.a11y-dyslexia`, `.a11y-reduce-motion`, `.a11y-forced-colors`).

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-case-navigator';

const navigator = document.createElement('ds-case-navigator');
navigator.setAttribute('current-index', '0');
navigator.setAttribute('total-cases', '5');

navigator.results = [
  { id: 'case-1', title: 'Agentic AI Design', snippet: 'DesignOps automation cuts delivery latency...' },
  { id: 'case-2', title: 'Design System Architecture', snippet: 'Universal web component design tokens...' }
];

navigator.addEventListener('ds-case-select', (e) => {
  console.log('Navigated to case index:', e.detail.index);
});

navigator.addEventListener('ds-search-select', (e) => {
  console.log('Selected search result:', e.detail.item);
});

document.body.appendChild(navigator);
```
