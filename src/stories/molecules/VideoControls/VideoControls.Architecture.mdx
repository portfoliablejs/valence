# `ds-video-controls`

The `ds-video-controls` component is an interactive **controlled** Molecule designed to render quality, highly accessible playback buttons for floating, compact overlay video systems. It aggregates custom `ds-button` Atoms (configured as `variant="floating"` and `icon-variant="fill"` switches) and nested `ds-tooltip` cues.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host binds attributes (`playing`, `muted`, `cc-enabled`, `speed`, `label-*`) downstream to swap control icons, toggle active highlighting, and configure translations.
* **Events Out (`ds-video-action`):** Selecting any control switch fires a unified custom event payload indicating the selected action name (`'play-pause'`, `'mute'`, `'cc'`, `'speed'`, or `'stop'`) upstream for video state management.

## Accessibility & Host ARIA Delegation

* **Button Delegation:** Replaces legacy markup with compliant `<ds-button>` wrappers, enabling native keyboard focus loops, hover scaling, and keyboard triggers.
* **Screen Reader Isolation:** Employs nested tooltip descriptions with keyboard shortcuts (`show-kbd="true"` and `kbd-label`). Screen readers are instructed to ignore background decorative graphics.
* **Zero i18n Localization:** Supports complete multi-language overrides by exposing custom `label-[control]` attributes (e.g., `label-play`, `label-cc-on`) directly.
* **Global Access Syncing:** MutationObserver on document roots reflects global setting cascades downstream as host element boolean flags:
  * `.a11y-dark-mode`: Swaps container background to matching dark player HUD tokens.
  * `.a11y-high-contrast`: Enforces strict AAA monochrome borders and outlines.
  * `.a11y-dyslexia`: Adjusts tooltip typeface families.
  * `.a11y-reduce-motion`: Disables hovering scale transitions.
  * `.a11y-forced-colors`: Employs active Windows system color maps.

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-video-controls';

const controls = document.createElement('ds-video-controls');
controls.setAttribute('playing', 'true');
controls.setAttribute('cc-enabled', 'true');
controls.setAttribute('speed', '1.5X');

// Customized translations
controls.setAttribute('label-play', 'Iniciar');
controls.setAttribute('label-pause', 'Pausar');

controls.addEventListener('ds-video-action', (e) => {
  console.log('Action selected:', e.detail.action);
});

document.body.appendChild(controls);
```
