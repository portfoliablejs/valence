# Changelog

<br />

## [1.0.0] - 2026-07-20

<br />

### Initial Release

The stable debut of the presentational video player control bar molecule (`ds-video-controls`) featuring custom floating button integrations, customizable speed rates, and full localization support.

### Added
- **Interactive Control Molecule:** Created `<ds-video-controls>` aggregating custom button and tooltip atoms.
- **Dumb Component Conformance:** Released of localized attribute translations for play, mute, speed, captions, and stop options.
- **Root Context Synchronization:** MutationObserver on document roots reflects global setting cascades downstream as host element boolean flags.
- **Thematic Overrides:** Bound mobile backdrop backgrounds, outlines, and drop shadows to `--ds-video-controls-*` properties.
- **Storybook Workspace:** Configured complete interactive Controls and detailed CSF3 stories inside `VideoControls.stories.js`.

### Fixed
- **Style Purging:** Deleted all legacy layout, SVG, and typography CSS, allowing children to inherit design variables effortlessly.
