## [1.1.0] - 2026-07-23

<br />

### Thumbnail Composition Update

This release updates `ds-gallery-item` to compose the shared `ds-thumbnail` atom instead of rendering a local image wrapper, and adds dedicated architecture/changelog documentation for maintainers.

<br />

### Summary of Changes

<table>
  <thead>
    <tr>
      <th>Classification</th>
      <th>Scope / Context</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`Changed`</td>
      <td>`GalleryItem.js`</td>
      <td>Replaced local thumbnail wrapper/image composition with `ds-thumbnail` (`custom-only`) as the render surface.</td>
    </tr>
    <tr>
      <td>`Changed`</td>
      <td>`gallery-item.css`</td>
      <td>Updated thumbnail styles to target the `ds-thumbnail` host and preserve hover, focus, and active interaction feedback.</td>
    </tr>
    <tr>
      <td>`Fixed`</td>
      <td>`locked visuals`</td>
      <td>Locked-state blur is now applied to the shared thumbnail host consistently with protected-card behavior.</td>
    </tr>
    <tr>
      <td>`Added`</td>
      <td>`component docs`</td>
      <td>Added architecture and changelog MDX coverage for `ds-gallery-item` to align with Valence molecule documentation standards.</td>
    </tr>
  </tbody>
</table>

<br />

### Added
- **Architecture Documentation:** Introduced `GalleryItem.Architecture.mdx` covering API, event contract, interaction model, and CSS hooks.
- **Changelog Coverage:** Added component-level release notes with scoped, classified updates.

### Changed
- **Thumbnail Surface:** Gallery cards now pass `thumb-src` into `ds-thumbnail` via `screen-image` and preserve ratio/height constraints.
- **Interaction Layer:** Existing click/keyboard select behavior remains on the same gallery thumbnail surface semantics.

### Fixed
- **Shared Thumbnail Consistency:** Gallery items now rely on the design system's thumbnail atom for image rendering behavior rather than duplicating local image mechanics.

## [1.0.0] - 2026-07-21

<br />

### Initial Release

Initial release of the `ds-gallery-item` molecule for case study card previews, lock-state redaction, and optional metadata pills.

<br />

### Added
- **Gallery Item Molecule:** Added `<ds-gallery-item>` with title, short description, read-time, lock icon, and pills support.
- **Selection Event:** Added `ds-case-select` custom event for host-level case navigation.
- **Protected State Support:** Added `is-protected` and `is-unlocked` behavior for locked/unlocked rendering.
