# `ds-gallery-item`

The `ds-gallery-item` component is a presentational Molecule that composes a clickable preview thumbnail (`ds-thumbnail`), metadata text, and optional action pills for portfolio and case study catalogs.

---

## Principles

### Unidirectional Data Flow
The component follows the **State In / Events Out** pattern:
* **State In (Attributes):** Host applications provide title, short description, read time, thumbnail source, thumbnail frame selectors (`thumb-category`, `thumb-brand`, `thumb-model`, `thumb-color`, optional `thumb-device-src`), frame ratio, and state flags (`is-protected`, `is-unlocked`, `has-video`, `has-repo`, `has-live`).
* **Events Out (`ds-case-select`):** The thumbnail activation surface dispatches selection intent via a bubbling/composed CustomEvent.
* **Derived Presentation State:** Lock/unlock visuals and pill visibility are derived from host-provided booleans only.

## Accessibility & Interaction

* **Keyboard Activation:** Selection surface supports `Enter` and `Space` key activation.
* **Focusable Surface:** Thumbnail host is exposed with `tabindex="0"` and `role="button"`.
* **Protected State Redaction:** Locked cards hide read time, blur the thumbnail surface, and replace title/description text.
* **Semantic Indicator:** Protected cards display lock-open / lock-closed iconography to communicate current availability.

## Integration Example

```javascript
import 'valence/src/stories/molecules/GalleryItem/GalleryItem.js';

const card = document.createElement('ds-gallery-item');
card.setAttribute('title', 'Agentic AI Design');
card.setAttribute('short-desc', 'Exploration of orchestration and UX.');
card.setAttribute('thumb-src', '/holofante.avif');
card.setAttribute('thumb-model', 'Apple iPhone 12');
card.setAttribute('thumb-color', 'Black');
card.setAttribute('read-time', '3 min');
card.setAttribute('has-video', '');
card.addEventListener('ds-case-select', () => {
  console.log('Open case details');
});

document.body.appendChild(card);
```
