# ds-navigation-menu

The ds-navigation-menu component is a controlled molecule for compact top-bar navigation actions. It composes icon buttons, tooltips, and contextual menus to expose language and accessibility controls while emitting semantic events to the host app.

---

## Principles

### Unidirectional Data Flow

State In / Events Out contract:

- State In: Host attributes configure labels, tooltip content, avatar source, disabled state, and accessibility labeling.
- Events Out: Interaction emits immutable custom events for menu triggers and item selection.
- Encapsulated State: Internal menu open state and menu item toggles are managed in-component and reported outward.

## Composition

The component has no slots. It composes existing system components inside its Shadow DOM:

- ds-button (icon variant, fill icons)
- ds-divider (vertical separator)
- ds-tooltip (button helper labels + kbd hints)
- ds-contextual-menu (language and accessibility menus)

### Internal Structure

1. Left profile group: circular avatar button + about tooltip.
2. Vertical separator: ds-divider orientation="vertical".
3. Right action group:
- Language icon button + tooltip + contextual menu.
- Accessibility icon button + tooltip + contextual menu.

## Accessibility

- ARIA labels are delegated to focusable internal controls.
- Menu tooltips are suppressed while corresponding contextual menu is open.
- Accessibility classes are reflected from document.documentElement via MutationObserver.
- Forced colors, high contrast, reduced motion, and dyslexia mode are supported by host-reflected attributes.

## Integration Example

```javascript
import 'valence/ds-navigation-menu';

const nav = document.createElement('ds-navigation-menu');
nav.setAttribute('language-tooltip', 'Idioma');
nav.setAttribute('accessibility-tooltip', 'Acessibilidade');

nav.addEventListener('ds-navigation-menu-language-select', (e) => {
  console.log('Language:', e.detail.id);
});

nav.addEventListener('ds-navigation-menu-accessibility-select', (e) => {
  console.log('Accessibility update:', e.detail);
});

document.body.appendChild(nav);
```
