# Changelog

## [1.0.0] - 2026-07-20

<br />

### Initial Release

The stable debut of the presentational settings list row component (`ds-item-row`) featuring nesting triggers for leading icons, shortcuts, and custom settings inputs (toggles, checks, and radios).

### Added
- **Settings Row Molecule:** Created `<ds-item-row>` built specifically for list navigations and dashboard popovers.
- **Form Widget Syncing:** Observed settings automatically instantiate, configure, and project child interactive components inside right-aligned slots.
- **Root Context Syncing:** MutationObserver on document roots reflects global setting cascades downstream as host element boolean flags.
- **Thematic Overrides:** Bound child width, height, and knob variables to strict sub-atomic properties.
- **Storybook Workspace:** Configured complete interactive Controls and detailed CSF3 stories inside `ItemRow.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Synchronized Storybook override controllers to output precise `--ds-check-color`, `--ds-radio-color`, and `--ds-toggle-checked-bg` custom styles.
