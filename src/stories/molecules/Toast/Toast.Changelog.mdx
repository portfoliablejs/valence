## [1.0.0] - 2026-07-20

<br />

### Initial Release

The stable debut of the presentational dynamic notification toast component (`ds-toast`) featuring configurable text buttons, dynamic padding alignments, and localized translation parameters.

### Added
- **Overlay Notification Molecule:** Created `<ds-toast>` serving as a floating contextual information overlay.
- **Robust i18n Compatibility:** Removed all hardcoded text strings in favor of configurable localization attributes with English as fallback.
- **Thematic Overrides:** Bound container background, text color, border radius, border-width, border-color, and padding dimensions to standard `--ds-toast-*` properties.
- **Storybook Workspace:** Configured complete interactive Controls and detailed CSF3 stories inside `Toast.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Synchronized Storybook override controllers to output precise `--ds-toast-*` custom styles.
