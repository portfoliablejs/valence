# `ds-toast`

The `ds-toast` component is an interactive, presentational **controlled** Molecule designed to render floating system notification banners and status updates. It includes optional left "Close" `ds-button` actions, optional right "Never show this again" text buttons, complete accessibility syncing, and robust localizable attribute controls.

---

## Principles

### Unidirectional Data Flow
The component strictly adheres to the **State In / Events Out** pattern:
* **State In (Attributes / Properties):** The host binds attributes (`visible`, `case-title`, `show-close`, `show-never-show`, `aria-label`, `label-*`) downstream to trigger entry/exit slide animations, inject message strings, and configure translation labels.
* **Events Out:** User clicks on the toast body, the close button, or the never-show button emit customized event streams prefixed with `ds-toast-` upstream.

## Accessibility & Host ARIA Delegation

* **Polite Announcements:** Employs an internal `role="status"` and `aria-live="polite"` element wrapper so assistive screen-reader software cleanly triggers notification announcements without interrupting user focus.
* **ARIA Host Scrubbing:** Automatically purges Light DOM `aria-label` custom attributes on host custom elements and projects them down onto the focusable checkbox.
* **Global Access Syncing:** Uses a `MutationObserver` on the HTML root element to capture visual preferences:
  * `.a11y-dark-mode`: Adjusts default track contrast and shifts unchecked knobs to dark grays.
  * `.a11y-high-contrast`: Enforces strict AAA binary layouts (solid black backdrops with bordered white outlines).
  * `.a11y-dyslexia`: Overrides track label typefaces with Comic Sans MS.
  * `.a11y-reduce-motion`: Forces immediate 0ms entry/exit translations.
  * `.a11y-forced-colors`: Employs Windows active system colors (`Canvas` and `CanvasText`).

## Examples

### Basic Integration Example

```javascript
import 'valence/ds-toast';

const toast = document.createElement('ds-toast');
toast.setAttribute('case-title', 'Resume reading: Agentic AI Design Study');
toast.setAttribute('show-close', 'true');
toast.setAttribute('show-never-show', 'true');

// Binding custom translations
toast.setAttribute('label-never-show', 'Não mostrar novamente');
toast.setAttribute('label-close', 'Fechar notificação');

toast.addEventListener('ds-toast-never-show', () => {
  localStorage.setItem('block-agentic-study-toast', 'true');
});

document.body.appendChild(toast);
```
