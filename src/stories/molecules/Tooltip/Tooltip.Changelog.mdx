# Changelog

<br />

## [1.0.0] - 2026-07-20

<br />

### Initial Release

The stable debut of the presentational descriptive overlay tooltip component (`ds-tooltip`) featuring automatic parent sibling anchoring, custom shortcut combinations with `ds-kbd`, and complete root visual contrast syncing.

### Added
- **Overlay Tooltip Molecule:** Created `<ds-tooltip>` serving as a floating contextual information overlay.
- **Auto-Anchoring Mechanics:** Passive hover/focus event listeners bound directly to parent triggers or preceding siblings.
- **Thematic Overrides:** Bound container background, border radius, border-width, and border-color to standard `--ds-tooltip-*` properties.
- **Storybook Workspace:** Configured complete interactive Controls and detailed CSF3 stories inside `Tooltip.stories.js`.

### Fixed
- **Sub-Atomic Story Mapping:** Synchronized Storybook override controllers to output precise `--ds-tooltip-*` custom styles.
